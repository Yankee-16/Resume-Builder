# Resume Builder

A simple resume builder made with React and Redux.

### Instructions to run

- Install dependencies

```bash
    npm install
```

or if you use yarn,

```bash
yarn install
```

- Run the development server

```bash
npm run start
```

or with yarn,

```bash
yarn start
```

- Build Production version

```bash
npm run build
```

or with yarn,

```bash
yarn build
```
